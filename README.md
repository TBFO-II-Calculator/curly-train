# Calculator

## Get started
- Go to folder bin/.
- execute file ```main```.
```
$ ./main

 ,-.     .         .     .           
/        |         |     |           
|    ,-: | ,-. . . | ,-: |-  ,-. ;-. 
\    | | | |   | | | | | |   | | |   
 `-' `-` ' `-' `-` ' `-` `-' `-' '   

Operator:
 ()	-- pengelompokan
  ^	-- perpangkatan
  *	-- perkalian
  /	-- pembagian
  +	-- penjumlahan/positif
  -	-- pengurangan/negatif
  !	-- keluar program
> 
```
- Enter arithmetic expression that you want to evaluate.
e.g.,
```
> 6.2^1^999
result: 6.2
> 
```
- Enter '!' to exit the program.
```
> !
$ 
```
## Compile.
```
$ gcc main.c -o main -lm
```
